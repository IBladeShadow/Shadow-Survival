# Shadow Survival

**A fast-paced 2D survival game written entirely in C++ using SDL2 – inspired by Terraria and Minecraft, but lighter and faster**

Survive in a ruthless world where every night gets darker and deadlier.  
Gather resources, craft, fight enemies that only come out at night, and become strong enough to see the sunrise.

### Key Features
- Infinite procedurally generated world with chunks
- Real day/night cycle + flashlight with battery
- Inventory, hotbar, and chest storage system
- 4 weapons: Sword, Pistol, Rifle, Shotgun
- Power potions: Strength, Speed, Armor
- Shop system with NPC and daily quests
- Full Persian (Farsi) and English language support
- Auto-save and customizable keybinds
